# Solventix Dental — Client Landing Site

A high-conversion landing page built for a boutique dental practice. Editorial design, cinematic scroll reveals, and a mobile-first layout built to convert visitors into booked appointments.

Live at: [solventixdental.vercel.app](https://solventixdental.vercel.app/)

---

## What Problem It Solves

Boutique dental clinics lose patients to weak digital presence. A generic WordPress template does not build trust. This site closes that gap — premium aesthetics and performance that signal quality before a patient ever calls.

---

## Features

- **Cinematic scroll animations** — GSAP-powered section reveals and parallax effects
- **Mobile-first responsive layout** — fully optimized from 320px to widescreen
- **Conversion-focused structure** — headline, social proof, services, and CTA above the fold
- **Fast load performance** — no framework overhead, vanilla HTML/CSS/JS
- **SEO ready** — semantic HTML, `robots.txt`, `sitemap.xml` included
- **Multi-format image assets** — WebP with JPG/PNG fallbacks for browser compatibility

---

## Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Semantic markup |
| CSS3 | Layout, animations, responsive design |
| JavaScript (ES6+) | Interaction and scroll behaviour |
| GSAP 3 | Scroll-triggered animations and parallax |
| Vercel | Deployment |

## Getting Started

No build step required. Open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
# or
python -m http.server 8000
```

---

## File Structure

```
├── index.html          # Main page
├── styles.css          # All styles and responsive breakpoints
├── script.js           # GSAP animations and interaction logic
├── robots.txt          # SEO directives
├── sitemap.xml         # Sitemap for search engines
└── *.webp / *.jpg      # Optimised image assets
```
